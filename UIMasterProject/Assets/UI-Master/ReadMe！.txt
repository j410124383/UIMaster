欢迎使用UImaster，这个组件可以帮助游戏制作者专注于游戏玩法本身。
将UI相关的工作交由我们来先完成。
制作者只需要替换对应的素材即可。

我们需要先准备一些操作：
1.	edit-project setting -text mesh pro -import TMP Essentials
	需要先将tmp的预设安装一下。

2. 在PackageManager 里面安装本地化组件，Localization.并在project setting - Localization中，将文件替换上去。

3. 在project setting-textmesh pro-settings中，将default sprite asset替换为spriteasset_default，	
	将default style sheet替换为stylesheet_Default，将default font asset 替换为notosansSC-Black SDF文件，
	当然后续也可以更加项目使用字体来替换。
